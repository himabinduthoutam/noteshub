# Noteshub

Noteshub is a Django-based web application that allows users to create, organize, edit, and manage personal notes efficiently in one place.

## Features
- Create notes
- Edit notes
- Delete notes
- Organize notes

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## Installation

git clone https://github.com/himabinduthoutam/noteshub.git
cd noteshub
pip install -r requirements.txt
python manage.py runserver
